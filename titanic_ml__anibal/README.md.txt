# Proyecto Titanic - Machine Learning

Este proyecto corresponde al desarrollo de un pipeline de analítica de datos aplicado al dataset Titanic.

## Contenido

- Exploración de datos (EDA)
- Limpieza y tratamiento de datos
- Comparación de frameworks (CRISP-DM, KDD, TDSP)
- Modelos de Machine Learning:
  - Regresión Logística
  - Random Forest
- Evaluación mediante métricas y curva ROC

## Resultados

- Accuracy Regresión Logística: ~0.80
- Accuracy Random Forest: ~0.82
- AUC: ~0.87
